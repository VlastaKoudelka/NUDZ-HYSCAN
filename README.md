## BVA export

The current version of the scripts work with exported dat from Brainvision Analyzer 2:
- gereric data export
 - Format: ASCII
 - Orientation: MULTIPLEXED
 - Line Delimiter: CRLF (PC style)
 - Write header file: yes
 - Write marker file: yes
